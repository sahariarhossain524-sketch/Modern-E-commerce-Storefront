# 🛍️ Modern E-commerce Storefront

A premium, highly responsive e-commerce storefront built to demonstrate advanced frontend development skills. Features a modern minimalist aesthetic, complex state management for a shopping cart, and fluid animations.

### 🔴 [Click Here to View Live Project](https://ecommerce-store-rust-psi.vercel.app)

![E-commerce Preview](https://images.unsplash.com/photo-1441986300917-64674bd600d8?auto=format&fit=crop&w=1600&q=80)

## ✨ Key Features

- **Dynamic Product Catalog**: Beautiful 4-column masonry-style grid featuring premium product cards with hover effects.
- **Global State Management**: Fully functional Shopping Cart (Add, Remove, Adjust Quantity) managed efficiently using **Zustand**.
- **Modern Animations**: Page transitions, scroll reveals, and interactive micro-animations powered by **Framer Motion**.
- **Responsive Design**: Flawless UI across Desktop, Tablet, and Mobile devices.
- **Premium UI/UX**: Built with Tailwind CSS, utilizing a clean monochrome color palette and modern typography.

## 🛠️ Technologies Used

- [Next.js](https://nextjs.org/) (App Router)
- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Zustand](https://docs.pmnd.rs/zustand/getting-started/introduction) (Cart State)
- [Framer Motion](https://www.framer.com/motion/) (Animations)
- [Lucide React](https://lucide.dev/) (Icons)
