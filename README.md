## 🚀 Features

- **Dynamic Cart Management**: Real-time cart state with interactive sidebars and persistent storage.
- **Framer Motion Animations**: Smooth page transitions, micro-interactions, and beautiful layout reveals.
- **Prisma ORM**: Robust database architecture and schema management for products and orders.
- **Ultra-Modern UI**: Premium storefront design with Tailwind CSS and custom design tokens.
- **Next.js App Router**: Optimized rendering, fast navigation, and SEO-friendly architecture.

## 💻 Tech Stack

| Technology | Description |
|------------|-------------|
| **Next.js** | React framework for production-grade applications |
| **Prisma** | Next-generation Node.js and TypeScript ORM |
| **Tailwind CSS** | Utility-first CSS framework for rapid UI development |
| **Framer Motion** | Production-ready animation library for React |
| **Lucide React** | Beautifully simple, pixel-perfect icons |

## 🛠️ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/sahariarhossain524-sketch/Modern-E-commerce-Storefront.git
   ```

2. **Navigate to the directory**
   ```bash
   cd Modern-E-commerce-Storefront
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Environment Variables**
   Set up your database connection string in `.env`:
   ```env
   DATABASE_URL="your_database_url_here"
   ```

5. **Run Prisma Migrations**
   ```bash
   npx prisma db push
   ```

6. **Start the development server**
   ```bash
   npm run dev
   ```
